# 🍽️ API de Gerenciamento de Restaurante

Esta API foi desenvolvida utilizando Node.js e SQLite, com o objetivo de gerenciar produtos e mesas de um restaurante de forma simples e eficiente.

## ⚙️ Funcionalidades

A API permite:

### ✅ Produtos

Criar produtos

Editar produtos

Remover produtos

Listar produtos

### ✅ Mesas

Listar mesas

Abrir mesas

Fechar mesas

Visualizar mesas indisponíveis

### ✅ Pedidos por mesa

Adicionar produtos a uma mesa

Listar produtos de uma mesa

Visualizar a quantidade total de itens

Visualizar o valor total a ser pago

## ▶️ Como executar o projeto

Para utilizar a API, siga os passos abaixo:

### 1️⃣ Clone o repositório

git clone <url-do-repositorio>

### 2️⃣ Instale as dependências

npm install

### 3️⃣ Execute o servidor em modo desenvolvimento

npm run dev

## 🧪 Como testar as requisições

Abra o aplicativo Insomnia

Importe o arquivo "Requests-Insomnia" presente no projeto

Após a importação, todas as rotas estarão disponíveis

Basta executar as requisições e utilizar as funcionalidades da API
